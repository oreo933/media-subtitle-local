# Changelog

All notable changes to this project will be documented in this file.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), adapted for the current stage of the project.

## [v0.1.0] - 2026-04-21

Initial public release.

### Added
- First public GitHub release: `v0.1.0`
- GitHub-style README with project positioning, highlights, FAQ, contribution notes, privacy notes, and English summary
- Public repository metadata including description and topics
- Local-first Windows desktop subtitle workflow documentation
- Release page for the first public version

### Changed
- Refined the public project introduction to better reflect the actual use case:
  - handling English / Japanese videos without ready-to-use subtitles
  - generating usable Chinese subtitles through a local workflow
- Reworked README structure for better public presentation and onboarding
- Improved repository presentation for open-source publishing

### Security
- Sanitized public-facing project materials to avoid exposing personal information
- Kept build artifacts, logs, local binaries, and model files outside the repository by default
